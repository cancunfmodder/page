NexUI5 1.1 · Samsung A41

A OneUI 5–based custom ROM for the Samsung A41, with a smoother OneUI 5.1 launcher, OneUI 5.1.1 clock, Zest Menu, custom boot logo, and root preinstalled. Beta and discontinued.

Device: Samsung A41 Base: OneUI 5 Status: Beta Development: Discontinued
Overview

NexUI5 1.1 is built on OneUI 5 for the Samsung A41. The focus is on a smoother launcher, a modern clock, and a clean UI while keeping the ROM usable for daily tasks (with some important bugs you must accept before flashing).

The project reached a beta stage and is now discontinued. The final beta is still available for users who understand the limitations.

Features
  • OneUI 5.1 launcher (smoother)
  • Clock from OneUI 5.1.1
  • Zest Menu
  • Custom boot logo
  • Root preinstalled
What works
  • System UI and general navigation
  • Wi‑Fi and mobile data
  • Audio (speaker, basic media playback)
Known broken
  • Bluetooth
  • Camera
  • Flash/torch
How to flash
  • 1. Flash Stock ROM
  • 2. Flash TWRP and VBMETA in AP
  • 3. Flash NexUI.zip
  • 4. Flash Kernel.zip
  • 5. Flash Guard.zip
  • 6. Format data and reboot
